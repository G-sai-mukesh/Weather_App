# 🌦 Premium Weather App

A modern full-stack weather application built using Node.js, Express.js, HTML, CSS, and JavaScript.

## 🚀 Live Demo

Frontend:
https://weather-app-omega-five-45.vercel.app/

Backend API:
https://weather-app-1-8yfo.onrender.com

---

# ✨ Features

- Current weather data
- 3-day weather forecast
- Auto location detection
- Premium modern UI
- Dynamic UI theme changes based on weather temperature
- Recent searches
- Popular Indian cities
- Responsive design
- Real-time weather updates

---

# 🛠 Tech Stack

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js
- Express.js

## API
- WeatherAPI.com

## Hosting
- Vercel (Frontend)
- Render (Backend)
