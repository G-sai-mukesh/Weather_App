# 🌦 Premium Weather App

A modern full-stack weather application built using Node.js, Express.js, HTML, CSS, and JavaScript.

## 🚀 Live Demo

Frontend:
https://g-sai-mukesh.github.io/Weather_App/client/index.html

Backend API:
https://weather-app-1-8yfo.onrender.com

---

# ✨ Features

- Current weather data
- 3-day weather forecast
- Auto location detection
- Indian city search suggestions
- Premium modern UI
- Recent searches
- Popular Indian cities
- Responsive design
- Real-time weather updates

# 🛠 Tech Stack

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js

- GitHub (Frontend)
